## 목표

마을에서 술집 버튼을 클릭하면 진입하는 용병 고용 화면의 기초 구조를 구현한다.

---

## 산출물

- `TavernScene.unity`
- 마을 건물 버튼 연결

---

## 요구사항

### 화면 구성

```
┌─────────────────────────────────────┐
│ [뒤로가기]      술  집       [골드] │
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────────────────────┐    │
│  │     고용 후보 용병 목록      │    │
│  │   (3명 표시)                │    │
│  │                             │    │
│  │  [용병1] [용병2] [용병3]    │    │
│  │                             │    │
│  └─────────────────────────────┘    │
│                                     │
│  ┌─────────────────────────────┐    │
│  │      선택 용병 상세 정보     │    │
│  │   이름 / 직업 / 스탯        │    │
│  │                             │    │
│  │   [고용하기: 100G]          │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
```

### 마을 버튼 추가

- 기존: 던전 입구, 길드 하우스
- 추가: 술집 버튼
