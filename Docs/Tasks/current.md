## 목표

마을에서 길드 하우스 버튼을 클릭하면 진입하는 용병 관리 화면의 기초 구조를 구현한다.

---

## 산출물

- `GuildHouseScene.unity`
- 마을 건물 버튼 연결

---

## 요구사항

### 화면 구성 (레이아웃)

```
┌─────────────────────────────────────┐
│ [뒤로가기]     길드 하우스     [골드] │
├─────────────────────────────────────┤
│                                     │
│  ┌──────────┐    ┌──────────────┐   │
│  │ 소유 용병 │    │  상세 정보   │   │
│  │  목록    │    │    패널     │   │
│  │          │    │              │   │
│  └──────────┘    └──────────────┘   │
│                                     │
│  ┌──────────────────────────────┐   │
│  │       파티 편성 슬롯 (4칸)     │   │
│  └──────────────────────────────┘   │
└─────────────────────────────────────┘
```

### 구현 방식 - **별도 씬**

- TownScene → GuildHouseScene 전환
- 독립적인 UI 관리

---

## 설계 가이드

### 씬 전환

```csharp
// TownScene에서
public void OnGuildHouseClick()
{
    GameManager.Instance.LoadScene("GuildHouseScene");
}
```

### 뒤로가기

```csharp
public void OnBackClick()
{
    GameManager.Instance.LoadScene("TownScene");
}
```
